GCode Table

The table outlines the GCode commands accepted by the CNC machine.

GCode Name Description
G00 Rapid Positioning Move each axis at its max speed until it reaches the positionit was trying to reach. Traditionally used to move the bit to astarting location or ending location.
G01 Linear Interpolation Move the tool from the current location to the given location in a linear path between the points by interpolating steps between the positions.
G02 Circular Interpolation, Clockwise Move the tool from the current location to the given location in a clockwise arc from another given center point by interpolating steps along the arc.
G03 Circular Interpolation, Counterclockwise Move the tool from the current location to the given location in a counterclockwise arc from another given center point by interpolating steps along the arc.
